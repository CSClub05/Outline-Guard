# Changelog

## 1.0.1

Minecraft 1.21.8 release.

- Updated the project to target Minecraft 1.21.8.
- Fixed a Minecraft 1.21.8 GUI crash caused by the marker screen requesting the vanilla background blur more than once in the same frame.
- Replaced the blurred GUI background with a simple dark overlay.
- Fixed the marker list display not refreshing immediately after using **Add marker**, **Set as only marker**, or **Remove marker**.

## 1.0.0

Initial release.

- Fully client-side Outline Guard behavior.
- GUI opened with D + C.
- Configurable marker blocks.
- Smooth mining pause/resume behavior.
- No server install required.
- No custom networking.
- No mixins.
